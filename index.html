الوحدة الأولى — الصفحة ٣

قراءة من الرسم + قيمة عظمى/صغرى + إكمال مربع
السؤال ١
رأس المنحنى من الرسم
الفكرة: رأس المنحنى هو أعلى/أدنى نقطة يغيّر عندها المنحنى اتجاهه.
  • نحدد نقطة القاع (لأن المنحنى مفتوح لأعلى في الرسم الأول).
  • نقرأ الإحداثي السيني ثم الصادي من الشبكة.
الرأس: (٢ ، −٣).
السؤال ٢
محور التماثل
الفكرة: محور التماثل خط رأسي يمر بالرأس، ومعادلته: س = (قيمة س عند الرأس).
بما أن الرأس عند س = ٢ ⇒ محور التماثل: س = ٢.
السؤال ٣
القيمة الصغرى من الرسم
الفكرة: إذا كان المنحنى يفتح لأعلى ⇒ أقل قيمة للدالة تكون عند الرأس.
قيمة ص عند الرأس = −٣ ⇒ القيمة الصغرى: ص = −٣.
طالما المنحنى مفتوح لأعلى ⇒ الصغرى عند رأس المنحنى.
السؤال ٤
قيمة عظمى
الدالة: y = 2 − (x + 3)²
الفكرة: لأن أمام التربيع إشارة سالبة ⇒ المنحنى يفتح لأسفل ⇒ أعلى قيمة عند الرأس.
  • (س + ٣)² ≥ ٠ دائمًا.
  • إذن −(س + ٣)² ≤ ٠ دائمًا، وأكبر قيمة لها = ٠.
  • أكبر قيمة لـ ص = ٢ + ٠ = ٢.
القيمة العظمى: ٢.
طالما المنحنى مفتوح لأسفل ⇒ العظمى عند رأس المنحنى.
السؤال ٥
قيمة صغرى
الدالة: y = (x + 2)² + 3
الفكرة: (… )² أصغر قيمة لها = ٠ ⇒ الصغرى تكون عند الرأس.
  • (س + ٢)² ≥ ٠ دائمًا.
  • أصغر قيمة عندما (س + ٢)² = ٠ ⇒ س = −٢.
  • إذن أصغر قيمة لـ ص = ٣.
القيمة الصغرى: ٣.
السؤال ٦
إيجاد الرأس (−ب ÷ ٢أ)
الدالة: y = x² − 4x + 1
  • أ = ١ ، ب = −٤.
  • س عند الرأس: −ب٢أ = −(−٤)٢×١ = ٤٢ = ٢
  • نعوض س = ٢: ص = ٤ − ٨ + ١ = −٣.
نقطة الرأس: (٢ ، −٣).
السؤال ٧
الرسم الثاني
الفكرة: نفس طريقة السؤال (١،٢،٣) ولكن من الرسم الثاني (المنحنى مفتوح لأسفل).
  • نحدد القمة (لأن المنحنى يفتح لأسفل).
  • القمة تقرأ من الشبكة: (١ ، ٢).
  • محور التماثل: س = ١.
  • القيمة العظمى: ص = ٢.
الرأس (١ ، ٢) — محور التماثل س = ١ — العظمى ص = ٢.
السؤال ٨
إكمال مربع
ص = س² + ٤س + ١
  • نصف معامل س = ٢.
  • س² + ٤س = (س + ٢)² − ٤.
  • إذن ص = (س + ٢)² − ٣.
ص = (س + ٢)² − ٣.
السؤال ٩
إكمال مربع
ص = س² − ٦س + ٥
  • نصف معامل س = ٣.
  • س² − ٦س = (س − ٣)² − ٩.
  • إذن ص = (س − ٣)² − ٤.
ص = (س − ٣)² − ٤.

الوحدة الأولى — الصفحة ٤

مميز + تقاطع + اختيار من متعدد
السؤال ١
رأس الدالة القياسية
الفكرة: إذا كانت ص = (س − م)² + ب ⇒ الرأس = (م ، ب).
م = ٢ ، ب = −٥ ⇒ م + ب = −٣.
السؤال ٢
نوع الجذور بالمميز
س² − ٣س − ١٠ = ٠
  • المميز Δ = ب² − ٤أج = (−٣)² − ٤(١)(−١٠) = ٩ + ٤٠ = ٤٩.
  • Δ > ٠ ⇒ جذران حقيقيان مختلفان.
الإجابة: حقيقيان مختلفان.
السؤال ٣
عدد الجذور
س² − ٢س + ٩ = ٠
  • Δ = (−٢)² − ٤(١)(٩) = ٤ − ٣٦ = −٣٢.
  • Δ < ٠ ⇒ لا يوجد جذور حقيقية.
عدد الجذور الحقيقية: ٠.
السؤال ٤
شرط عام
إذا كان Δ = ب² − ٤أج < ٠ ⇒ لا توجد جذور حقيقية ⇒ عدد الجذور = ٠.
السؤال ٥
جذران متساويان
س² − ك س + ٩ = ٠
  • جذران متساويان ⇒ Δ = ٠.
  • Δ = ك² − ٤(١)(٩) = ك² − ٣٦ = ٠ ⇒ ك² = ٣٦.
ك = ±٦.
السؤال ٦
حل نظام (مستقيم + دائرة)
ص = س + ١ ، و س² + ص² = ١٣
  • نعوض ص = س + ١ في الدائرة: س² + (س+١)² = ١٣.
  • ٢س² + ٢س + ١ = ١٣ ⇒ ٢س² + ٢س − ١٢ = ٠ ⇒ س² + س − ٦ = ٠.
  • (س − ٢)(س + ٣) = ٠ ⇒ س = ٢ أو س = −٣.
  • إذا س = ٢ ⇒ ص = ٣. وإذا س = −٣ ⇒ ص = −٢.
يوجد حلان، ومن الخيارات الصحيحة: (٢ ، ٣).
السؤال ٧
حل نظام
ص = س − ١ ، و س² + ص² = ٥
  • س² + (س−١)² = ٥ ⇒ ٢س² − ٢س + ١ = ٥ ⇒ ٢س² − ٢س − ٤ = ٠ ⇒ س² − س − ٢ = ٠.
  • (س − ٢)(س + ١) = ٠ ⇒ س = ٢ أو س = −١.
  • إذا س = ٢ ⇒ ص = ١ (حل صحيح وموجود في الخيارات).
من الخيارات: (٢ ، ١).
السؤال ٨
إيجاد م وب
إذا كانت (٢ ، ١) تحقق: ص = س − م ، و س² + ص² = ب
  • ١ = ٢ − م ⇒ م = ١.
  • ب = ٢² + ١² = ٤ + ١ = ٥.
م + ب = ١ + ٥ = ٦.
السؤال ٩
ملاحظة على خطأ محتمل في الورقة
إذا كانت (٣ ، ٢) تحقق: ص = س − م ، و ب = ص² − س²
  • ٢ = ٣ − م ⇒ م = ١.
  • ب = ٢² − ٣² = ٤ − ٩ = −٥.
  • إذن م + ب = ١ + (−٥) = −٤.
الناتج −٤. (إذا لم يظهر ضمن الاختيارات في نسختك، فهذا غالبًا خطأ مطبعي في السؤال/الخيارات).
السؤال ١٠
عدد نقاط التقاطع
المستقيم ص = ٢س + ٣ ، والمنحنى ص = س²
  • نساوي: س² = ٢س + ٣ ⇒ س² − ٢س − ٣ = ٠.
  • (س − ٣)(س + ١) = ٠ ⇒ س = ٣ أو س = −١.
إذن يتقاطعان في نقطتين.
السؤال ١١
اختيار الرسم الصحيح (نوع الجذور)
الفكرة: جذران حقيقيان مختلفان ⇐ المنحنى يقطع محور السينات مرتين.
اختر الرسم الذي يقطع محور السينات مرتين (في الورقة هو المنحنى الأزرق).
السؤال ١٢
اختيار الرسم (جذران متساويان)
الفكرة: جذران متساويان ⇐ المنحنى يلامس محور السينات مرة واحدة فقط.
اختر الرسم الذي يلامس محور السينات مرة واحدة (في الورقة هو المنحنى الأخضر).

الوحدة الأولى — الصفحة ٥

صح/خطأ + مميز + إكمال مربع + نوع الجذور
السؤال ١
صح / خطأ من الرسم
الدالة (حسب الرسم): ص = س² − ٤س + ٣
  • نقطة التحول ليست (١ ، ٢)، لأن س الرأس = −ب/٢أ = ٤/٢ = ٢ ثم ص(٢)=٤−٨+٣=−١ ⇒ الرأس (٢ ، −١) ⇒ العبارة خطأ.
  • محور التماثل س = ٢ ⇒ صح.
  • جذرا ص=٠: س² −٤س +٣ = (س−١)(س−٣) ⇒ ١ و ٣ ⇒ صح.
  • التقاطع مع محور الصادات: عند س=٠ ⇒ ص=٣ ⇒ (٠ ، ٣) ⇒ صح.
النتيجة: (١) خطأ — (٢) صح — (٣) صح — (٤) صح.
السؤال ٢
عدد الحلول
٢س² − ٥س + ٢ = ٠
  • Δ = (−٥)² − ٤(٢)(٢) = ٢٥ − ١٦ = ٩ > ٠.
عدد الحلول الحقيقية: ٢.
السؤال ٣
عدد الحلول
٤س² − ١٢س + ٩ = ٠
  • Δ = (−١٢)² − ٤(٤)(٩) = ١٤٤ − ١٤٤ = ٠.
عدد الحلول الحقيقية: ١ (جذر مكرر).
السؤال ٤
مميز = ٨
٤س² − ٦س + ك = ٠ ، والمميز Δ = ٨
  • Δ = (−٦)² − ٤(٤)ك = ٣٦ − ١٦ك = ٨.
  • ١٦ك = ٢٨ ⇒ ك = ٧٤
ك = ٧/٤.
السؤال ٥
مميز = ٤
ك س² − ٦س + ١ = ٠ ، والمميز Δ = ٤
  • Δ = (−٦)² − ٤(ك)(١) = ٣٦ − ٤ك = ٤.
  • ٤ك = ٣٢ ⇒ ك = ٨.
ك = ٨.
السؤال ٦
إكمال مربع (نقاط الفراغ)
  • س² − ٤س + ٥ = (س − ٢)² + ١
  • س² + ٦س + ٣ = (س + ٣)² − ٦
  • س² − ٥س + ٧ = (س − ٥٢ )² + ٣٤
السؤال ٧
إكمال مربع + عناصر الرسم
ص = س² − ٢س − ٣
  • نصف معامل س = ١.
  • س² − ٢س = (س − ١)² − ١.
  • إذن ص = (س − ١)² − ٤.
الصورة القياسية: ص = (س − ١)² − ٤
  • نقطة التحول: (١ ، −٤)
  • محور التماثل: س = ١
  • القيمة الصغرى: −٤
  • نقاط التقاطع مع السينات: بحل س² − ٢س − ٣ = ٠ ⇒ (س−٣)(س+١)=٠ ⇒ (−١ ، ٠) و (٣ ، ٠)
  • نقطة التقاطع مع الصادات: عند س=٠ ⇒ ص=−٣ ⇒ (٠ ، −٣)
الرسم للتوضيح: المنحنى مفتوح لأعلى والرأس عند (١ ، −٤).
السؤال ٨
نوع الجذور
  • س² − ٦س + ٩ = ٠ ⇒ Δ = ٠ ⇒ حقيقيان متساويان.
  • س² + ٥س + ٦ = ٠ ⇒ Δ = ٢٥ − ٢٤ = ١ > ٠ ⇒ حقيقيان مختلفان.
  • س² − ٣س = ٠ ⇒ س(س−٣)=٠ ⇒ حقيقيان مختلفان.
السؤال ٩
جذران متساويان
س² − ك س + ١ = ٠ (جذران حقيقيان متساويان)
  • شرط التساوي: Δ = ٠ ⇒ ك² − ٤ = ٠ ⇒ ك = ±٢.
ك = ±٢.

السؤال ١
جذران متساويان
س² − ٨س + ك = ٠
  • Δ = (−٨)² − ٤(١)ك = ٦٤ − ٤ك.
  • للجذرين المتساويين: Δ = ٠ ⇒ ٦٤ − ٤ك = ٠ ⇒ ك = ١٦.
ك = ١٦.
السؤال ٢
حل متباينة
س² − ٣س − ١٠ < ٠
  • نحل المعادلة: س² − ٣س − ١٠ = ٠ ⇒ (س−٥)(س+٢)=٠ ⇒ الجذور: −٢ ، ٥.
  • لأن أ موجب ⇒ التعبير سالب بين الجذرين.
الحل: −٢ < س < ٥.
السؤال ٣
حل متباينة
س² − س − ١٢ ≥ ٠
  • (س−٤)(س+٣) ≥ ٠.
  • الحل خارج الجذرين: س ≤ −٣ أو س ≥ ٤.
س ≤ −٣ أو س ≥ ٤.
السؤال ٤
حل متباينة
س² − ٣س < ٠
  • س(س−٣) < ٠ ⇒ الحل بين الجذرين.
٠ < س < ٣.
السؤال ٥
متباينة دائمًا صحيحة
س² − ٦س + ٩ ≥ ٠
  • س² − ٦س + ٩ = (س−٣)² ≥ ٠ دائمًا.
الحل: جميع قيم س الحقيقية.
السؤال ٦
لا يوجد حل حقيقي
س² < ٣س − ٥
  • ننقل: س² − ٣س + ٥ < ٠.
  • Δ = (−٣)² − ٤(١)(٥) = ٩ − ٢٠ = −١١ < ٠، ومع أ موجب ⇒ التعبير موجب دائمًا.

السؤال ٧
حل متباينة
س² ≤ ١٢س − ٣٦
  • س² − ١٢س + ٣٦ ≤ ٠ ⇒ (س−٦)² ≤ ٠.
  • مربع ≤ ٠ يتحقق فقط عندما يساوي صفر.
س = ٦ فقط.
السؤال ٨
حل بالإكمال إلى مربع
(١) س² + ٦س = −١٠
  • نصف معامل س = ٣ ⇒ س² + ٦س = (س+٣)² − ٩.
  • (س+٣)² − ٩ = −١٠ ⇒ (س+٣)² = −١.
لا يوجد حل حقيقي (لأن مربع عدد لا يساوي عددًا سالبًا).
(٢) س² − ٥س + ١ = ٠
  • نصف معامل س = ٥٢
  • س² − ٥س = (س − ٥/٢)² − (٥/٢)².
  • (س − ٥/٢)² − ٢٥/٤ + ١ = ٠ ⇒ (س − ٥/٢)² = ٢١/٤.
  • س = ٥/٢ ± √٢١ ٢
س = ٥ ± √٢١٢
السؤال ٩
صيغة تربيعية (تقريب)
(١) ٢س² + ٥س = −٦
  • نحوّل: ٢س² + ٥س + ٦ = ٠.
  • Δ = ٢٥ − ٤(٢)(٦) = ٢٥ − ٤٨ = −٢٣ < ٠.
لا يوجد حل حقيقي. (الجذور مركبة).
(٢) س² + ٣س − ٥ = ٠
  • س = −٣ ± √٢٩٢
  • √٢٩ ≈ ٥٫٣٩
س ≈ ١٫١٩ أو −٤٫١٩
السؤال ١٠

  • ٣س² + ١٠س − ٢ = ٠ ⇒ Δ = ١٠٠ + ٢٤ = ١٢٤ > ٠ ⇒ حقيقيان مختلفان.
